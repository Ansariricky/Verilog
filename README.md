# Verilog Hardware Projects

This repository contains a collection of my **Verilog-based digital design projects**.
Each folder represents a separate hardware design project implemented, simulated, and documented using Verilog HDL.

---



Each project generally follows this structure:

```
project-name
│
├── src        → Verilog design modules
├── tb         → Testbenches for simulation
├── docs       → Architecture diagrams and design notes
├── waveform   → Simulation waveform screenshots
└── README.md  → Project documentation
```

---

## Current Projects

### 1. 8-bit ALU (Arithmetic Logic Unit)

A simple 8-bit ALU implemented in Verilog supporting multiple arithmetic and logical operations.

**Features**

* Addition
* Subtraction
* Bitwise AND
* Bitwise OR
* Bitwise XOR
* Bitwise NOT
* Left Shift
* Right Shift

This project demonstrates:

* Combinational logic design
* Case-based control logic
* Verilog testbench creation
* Simulation and waveform analysis
